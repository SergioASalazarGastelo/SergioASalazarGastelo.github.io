Syntactic sugar en Ruby

En Ruby, todos los operadores son métodos, así por ejemplo es válido escribir

5.== 5  //con punto

5 == 5 //sin punto

5.send(:==,5) //usando el método "send"

las tres sentencias devuelven true, pues 5 es igual a 5, sin embargo sintácticamente se pueden escribir de esas tres formas. El método send, sirve para llamar a métodos dentro de un objeto cualquiera.

Otros ejemplos:

x[3] 
x.[] 3

x[3] = 'a' 
x.[]= 3, 'a' //en este caso el método se llama '[]='
